### Styleguide-Übersicht

#### Wolf Schneider
```
Titel: Klares, prägnantes Deutsch
Beschreibung: Verständliche Texte nach den bewährten Prinzipien von Wolf Schneider. 
Hauptsachen in Hauptsätze, Verben statt Substantive, konkret statt abstrakt.

Ideal für: Geschäftstexte, E-Mails, Berichte
Schwierigkeit: Einfach
```

#### Gendergerechte Sprache (Sternchen)
```
Titel: Inklusive Sprache mit Asterisk
Beschreibung: Moderne gendergerechte Formulierungen mit Sternchen-Notation. 
Student*innen, Mitarbeiter*innen, Professor*innen - für zeitgemäße Kommunikation.

Ideal für: Stellenausschreibungen, offizielle Kommunikation
Schwierigkeit: Einfach
```

#### Gendergerechte Sprache (Neutral)
```
Titel: Inklusive Sprache ohne Sonderzeichen
Beschreibung: Gendergerechte Formulierungen durch neutrale Begriffe und Umschreibungen. 
Studierende, Fachkräfte, Lehrende - elegant und barrierefrei.

Ideal für: Konservative Umgebungen, internationale Kommunikation
Schwierigkeit: Mittel
```

#### Gute Präsentationen
```
Titel: Überzeugende Präsentationen
Beschreibung: Strukturierte Folien nach dem SCS-Muster und MECE-Prinzip. 
Slide-Titel als Aussagen, klare Argumentation, wirkungsvolle Botschaften.

Ideal für: Business-Präsentationen, Pitches, Vorträge
Schwierigkeit: Mittel
```

#### Gene Zelazny Charts
```
Titel: Effektive Datenvisualisierung
Beschreibung: Professionelle Charts und Diagramme nach bewährten Prinzipien. 
Die richtige Chart-Art für jede Botschaft, klare Gestaltung, überzeugende Datengeschichten.

Ideal für: Berichte, Analysen, Dashboards
Schwierigkeit: Fortgeschritten
```

## Anwendungsseite

### Tab-Inhalte

#### Claude Projects
```
Überschrift: Styleguides in Claude Projects verwenden

Schritt 1: Projekt erstellen
1. Gehen Sie zu claude.ai und melden Sie sich an
2. Klicken Sie auf "New Project" 
3. Geben Sie einen Namen ein: "Deutsche Styleguides"
4. Wählen Sie "Add content"

Schritt 2: Styleguide hinzufügen
Wählen Sie eine Methode:

Option A - Datei hochladen:
• Laden Sie die .md-Datei herunter
• Drag & Drop in das Claude Project
• Claude erkennt automatisch die Formatierung

Option B - Text kopieren:
• Klicken Sie "📋 Regeln kopieren" 
• Fügen Sie den Text in "Project Instructions" ein
• Speichern Sie das Projekt

Schritt 3: Verwenden
• Starten Sie eine neue Unterhaltung im Projekt
• Claude hat automatisch Zugriff auf alle Styleguides
• Referenzieren Sie spezifische Regeln in Ihren Prompts

💡 Tipp: Kombinieren Sie mehrere Guides in einem Projekt für maximale Flexibilität.
```

#### ChatGPT
```
Überschrift: Styleguides mit ChatGPT verwenden

Option 1 - Custom GPT erstellen (ChatGPT Plus)
1. Gehen Sie zu chat.openai.com
2. Klicken Sie "Create a GPT"
3. Name: "Deutscher Stilberater"
4. Beschreibung: "Hilft bei der Erstellung deutscher Texte nach professionellen Styleguides"
5. Kopieren Sie unsere Konfiguration:

[Hier Custom GPT Konfiguration einfügen]

Option 2 - In normaler Unterhaltung
1. Kopieren Sie den gewünschten Styleguide
2. Beginnen Sie Ihre Unterhaltung mit:
   "Verwende folgende Stilrichtlinien: [eingefügter Guide]"
3. Stellen Sie Ihre Anfrage

Option 3 - Systemanweisung (ChatGPT Plus)
1. Gehen Sie zu den Einstellungen
2. Unter "Custom Instructions"
3. Fügen Sie eine Kurzversion des Guides ein
4. Wird automatisch bei jeder Unterhaltung angewendet

💡 Tipp: Referenzieren Sie spezifische Regeln direkt: "Schreibe nach Wolf Schneider Prinzip 3..."
```

#### Andere Tools
```
Überschrift: Weitere KI-Tools

Gemini (Google):
• Kopieren Sie den Styleguide als ersten Prompt
• Verwenden Sie "Angeheftete Unterhaltungen" für Wiederverwendung

Perplexity:
• Guides als Quellentext behandeln
• "Schreibe basierend auf diesen Regeln..." verwenden

Microsoft Copilot:
• In Office-Anwendungen als Kommentar einfügen
• "Schreibe E-Mail nach Wolf Schneider Regeln"

Notion AI / Obsidian:
• Guides als Template-Notiz speichern
• Bei Bedarf kopieren und referenzieren

💡 Allgemeiner Tipp: Die meisten KI-Tools verstehen deutsche Styleguide-Regeln, 
wenn sie klar strukturiert präsentiert werden.
```

#### Kombinationen
```
Überschrift: Mehrere Guides kombinieren

Beliebte Kombinationen:

Wolf Schneider + Gendergerecht (neutral)
Ideal für: Offizielle Geschäftskommunikation
Prompt: "Schreibe nach Wolf Schneider Prinzipien (klar, kurz) und verwende 
gendergerechte Sprache ohne Sternchen."

Präsentationen + Gene Zelazny  
Ideal für: Datengetriebene Präsentationen
Prompt: "Erstelle Präsentation nach SCS-Muster mit professionellen Charts 
nach Zelazny-Prinzipien."

Alle Guides zusammen
Ideal für: Umfassende Texte
Prompt: "Verwende alle verfügbaren Styleguides je nach Kontext: Wolf Schneider 
für Klarheit, Gendergerecht für Inklusion, Präsentationsregeln für Struktur."

Eigene Kombinationen:
1. Identifizieren Sie Ihre Anforderungen
2. Wählen Sie 2-3 passende Guides
3. Erwähnen Sie alle in Ihrem Prompt
4. Lassen Sie die KI automatisch das richtige Maß finden

💡 Tipp: Zu viele gleichzeitige Regeln können sich widersprechen. 
Starten Sie mit 1-2 Guides und erweitern Sie nach Bedarf.
```

## Über-Seite

### Projekthintergrund
```
Überschrift: Über KI-Styleguides

Das Projekt:
KI-Styleguides entstand aus der Erfahrung, dass KI-Tools zwar technisch 
beeindruckend sind, aber ohne klare Anweisungen oft generische oder inkonsistente 
Texte produzieren. Professionelle Styleguides lösen dieses Problem.

Unser Ansatz:
• Bewährte Regeln (Wolf Schneider, Gene Zelazny) für die digitale Ära
• Spezifisch für deutsche Sprache optimiert  
• Praktische Anwendung mit modernen KI-Tools
• Open Source für maximale Verbreitung

Die Guides basieren auf jahrzehntelang erprobten Prinzipien von Kommunikations-
experten und werden kontinuierlich für den Einsatz mit KI-Systemen weiterentwickelt.
```
